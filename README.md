# 경북대학교 K-Digital Training

본 교육 자료는 경북대학교 데이터융복합연구원이 운영하는 K-Digital Training 교육과정을 위해 제작하였습니다.

각 모듈은 이론과 실습을 기반으로 진행하며, 각 모듈별로 미니 프로젝트를 통해 학습 내용을 종합적으로 적용합니다.

## Module 12. 전이 학습 기반 모델

다양한 사전학습 LLM 모델을 활용하고, LLM 모델을 미세조정(Fine-Tuning)해보는 실습을 합니다.

## Module 13. 웹 기반 AI 서비스

Flask와 FastAPI를 학습하고, Streamlit을 이용하여 생성형 AI 기반 웹 서비스를 구현합니다.

## Module 14. 클라우드 기반 AI 서비스

미세조정한 모델과 웹 기반 AI 서비스를 클라우드 환경에서 배포하고 운영하는 방법을 학습합니다.
